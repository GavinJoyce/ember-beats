ember-beats
============

A drum machine built with Ember.js

https://www.youtube.com/watch?v=zvKVmZXqico

![screenshot](https://f.cloud.github.com/assets/2526/986155/c084c8b4-08d2-11e3-933e-45a5a5e3cab1.png)

Setup:

 - Install Node.js
 - ```npm install -g brunch@1.7.0```
 - ```npm install```
 - ```brunch watch -s```
